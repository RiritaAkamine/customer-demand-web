# Frontend

Next.js dashboard for Customer Demand Analyzer.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

The dashboard expects the backend API to run at `http://127.0.0.1:8000` by default. Copy `.env.local.example` to `.env.local` to override `NEXT_PUBLIC_API_BASE_URL`.

See the project-level `README.md` for the full setup guide and architecture notes.
