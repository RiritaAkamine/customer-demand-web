# 接客支援AIシステム

> 表情・音声・発話内容を統合的に解析し、顧客の関心度や心理状態をリアルタイムで推定するAI接客支援システム

## 概要

本システムは、顧客の表情・声のトーン・会話内容をリアルタイムで解析し、顧客の関心度や購買意欲を推定するとともに、販売員が次に取るべき接客アクションを提案するマルチモーダルAIシステムです。

従来のテキスト中心の顧客分析では把握しきれない非言語情報に着目し、画像認識、音声信号処理、音声認識、大規模言語モデル（LLM）を組み合わせることで、より人間に近い顧客理解を実現することを目指しました。

---

# 開発背景

## 非言語情報を活用した顧客理解への挑戦

人間同士のコミュニケーションでは、会話内容だけでなく、表情や声のトーン、話す速度などの非言語情報が重要な役割を果たしています。

近年は自然言語処理技術が発展していますが、テキストだけでは顧客の本音や感情の変化を十分に捉えることができません。

そこで本プロジェクトでは、言語情報だけでなく視覚情報・聴覚情報も統合的に扱うマルチモーダルAIに着目しました。

## 家電量販店での接客経験

私は家電量販店エディオンでのアルバイトを通じて、光回線やスマートフォンの乗り換え提案業務を経験しました。

接客の現場では、お客様が興味を持っているように見えても実際は警戒していたり、逆に無表情であっても強い関心を示していたりすることがあります。

熟練した販売員は、お客様の声のトーンや一瞬の表情変化を読み取りながら提案内容を調整しています。しかし、その判断は経験に依存する部分が大きく、新人販売員が短期間で身につけることは容易ではありません。

そこで、「熟練販売員の観察力をAIによって支援できないか」という発想から本システムを開発しました。

## 高校時代の開発経験とのつながり

高校時代にはOpenCVを用いて、笑顔の度合いを数値化し、その値によってAIとじゃんけんを行う画像認識システムを制作しました。

本プロジェクトは、その表情認識技術を発展させ、大学で学んだ音声信号処理や機械学習、さらにLLMによる推論技術を組み合わせることで、実社会の課題解決へ応用した発展的な研究・開発成果です。

# 主な機能

## 表情解析

* カメラ映像から顔領域を検出
* DeepFaceによる感情推定
* 7種類の感情をリアルタイム分析

対象感情

* Happy
* Neutral
* Sad
* Angry
* Surprise
* Fear
* Disgust

## 音声解析

* RMS（音量）
* Pitch（声の高さ）
* 発話検出

によって顧客の熱量や関心度を分析します。

## 音声認識

Whisper Large V3を利用してリアルタイムに文字起こしを実施します。

## LLMによる接客支援

以下の情報を統合して推論を実行します。

* 表情情報
* 音声特徴量
* 発話内容

LLMは

* 関心度
* 購買意欲
* 顧客心理
* 推奨接客アクション

を出力します。
